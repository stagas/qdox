
# qdox

shell docs using dox

## install

`npm install -g qdox`

## usage

```
  Usage: qdox [options] <file>                        
                                                      
  Options:                                            
                                                      
    -h, --help             output usage information   
    -V, --version          output the version number  
    -t, --type [type]      type of output             
    -d, --dev              shows private              
    -s, --search [string]  search string              
    -F, --file [name]      file to read
```
